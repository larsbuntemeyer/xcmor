Welcome to ``xcmor``
========================

``xcmor`` allows for lazy in-memory cmorization with `xarray <https://docs.xarray.dev/>`_.

Installing
----------

``xcmor`` can be installed using ``pip``

    >>> pip install xcmor


or using ``conda``

    >>> conda install -c conda-forge xcmor
